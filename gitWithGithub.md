### 1.commit后修改回到某个ish位置，连同硬盘（本地修改）
git reset --hard <commit_ish>  
### 2.修改HEAD，提交记录变，但文件不变（本地未修改）默认！
git reset --mixed <commit_ish>  
### 3.soft相当于mixed方法后再git add
git reset --soft <commit_ish>
